# Homebrew Tap for Elnora CLI

Homebrew formula for installing the [Elnora CLI](https://github.com/Elnora-AI/elnora-cli).

## Install

```bash
brew install elnora-ai/cli/elnora
```

## Usage

```bash
elnora auth login
elnora projects list
elnora doctor
```
